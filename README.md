# API Key Manager - macOS Menu Bar

A lightweight macOS menu bar app for managing your API keys.

## Features

- Menu bar icon with quick access
- Add, view, and delete API keys
- Copy keys to clipboard with one click
- Secure local storage
- Minimal, native macOS design

## Requirements

- macOS 13.0+
- Swift 5.9+
- Xcode 15+

## Build & Run

```bash
swift build
swift run
```

Or open in Xcode and press ⌘R.

## Usage

1. Click the key icon in the menu bar
2. Click "+" to add a new API key
3. Enter name, provider, and key
4. Hover over a key to copy or delete
